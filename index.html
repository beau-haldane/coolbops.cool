About these backgrounds...

Most of the background tiles in the Artistic Style Full Page Tiles section were not created with the intent of displaying text directly on the background image. There are too many contrasts, making the text hard to read.

Using a table centered on the page, and using a solid background color for the table, allows you to use colorful backgrounds without compromising the usability of the content.
